# Spell Checker Project

## Overview
This project is a simple Spell Checker Application developed in C using Hash Tables and Trie Data Structures for efficient word searching and suggestions.

The application allows users to:
- Check if a word exists
- Add words to the dictionary
- Remove words from the dictionary
- Get word suggestions
- Display words in descending order
- Find the largest word in the dictionary

---

# Features

- Fast word lookup using Hash Tables
- Prefix-based suggestions using Trie
- Case-insensitive word checking
- Add and remove dictionary words
- Word suggestions using entered letters
- Display dictionary in descending order
